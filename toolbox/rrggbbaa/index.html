#RRGGBBAA table

CSS Color Module Level 4 introduces new 8 (#rrggbbaa) and 4 (shortened #rgba) character hex notation for color to include the opacity level.

background: rgba(0,0,0,.2); // rgba()
background: #00000033;      // #rrggbbaa format
background: #0003;          // #rgba format
